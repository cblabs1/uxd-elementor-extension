== Installation ==

**Automatic Installation:**

1. Go to your WordPress admin dashboard
2. Navigate to Plugins > Add New
3. Search for "UXD Elementor Extension"
4. Click "Install Now" and then "Activate"

**Manual Installation:**

1. Download the plugin zip file
2. Go to Plugins > Add New > Upload Plugin
3. Choose the zip file and click "Install Now"
4. Activate the plugin after installation

**Requirements:**

* WordPress 5.0 or higher
* Elementor (free version) installed and activated
* PHP 7.4 or higher
* WooCommerce (for premium product widgets only)

== Frequently Asked Questions ==

= What widgets are included for free? =

The free version includes the powerful **Gallery Grid widget** with full functionality including lightbox, search, grid/masonry layouts, and responsive controls.

= What do I get with the premium version? =

Premium includes Product Carousel, Product Grid, Taxonomy Accordion, Post Grid widgets, WooCommerce integration, advanced styling options, and priority support.

= Is there a trial available? =

No trial is available. However, the free Gallery Grid widget gives you a good taste of our code quality and design standards.

= Is WooCommerce required? =

WooCommerce is only required for the premium Product Carousel and Product Grid widgets. All other widgets work independently.

= Does this work with any theme? =

Yes, UXD Elementor Extension works with any WordPress theme that supports Elementor. The widgets inherit your theme's styling while providing additional customization options.

= Can I upgrade later? =

Absolutely! You can upgrade to premium at any time from your WordPress admin dashboard. Your free Gallery Grid widget will continue to work seamlessly.

= Is there documentation available? =

Yes, comprehensive documentation is available at https://uxdesignexperts.com with detailed guides and video tutorials.
